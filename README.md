Network project simulating 

- Auto connect to router/switch in network (netmiko)
- Auto check network device "show interfaces"
- Auto parse (regex) for CRC errors and Input errors for each Network interface
- Auto fix using netmiko send cmds "shutdown" and "no shutdown"
- Show summary of device state and fixes

Modules used: Regex, Netmiko

Technologies used: Python, Cisco Packet Tracer, Ansible
